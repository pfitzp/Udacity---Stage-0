Getting Started With HTML - Notes

Basics of the Web
The web is accessed on a Computer/Web Browser, which communicates through the internet via HTTP protocol to access files hosted on servers. The majority of files hosted on these servers are HTML files or HyperText Markup Language.

HTML Basics
HTML files can contain four different things.

- Text content - What you see
- Markup - What it looks like
- References to other documents - Images, Videos, etc
- Links to other pages

Markup - Tags & Elements
HTML Markup is made up of Tags & Elements, which determines how your page looks. Each element generally contains an opening and closing tag, which surround content. Tags such as img (Images) and br (Break) do not contain a closing tag, and are called void tags. A good resource for all HTML tags and elements is this page.

HTML elements are either inline or block. A block element means the contents are all within an invisible block.

Document Structure
Making your code easy to read is a very important part of good programming, and making sure your code has good structure and indenting is a very big part of that.
